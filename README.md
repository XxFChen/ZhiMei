# Zhi Mei's Academic Website

This is the personal academic website for **Zhi Mei**, Marie Sklodowska-Curie Postdoctoral Fellow in Environmental Microbiology at the Catalan Institute for Water Research (ICRA), Girona, Spain.

## About

This website showcases:
- Research interests in environmental antimicrobial resistance (AMR) and antibiotic resistance genes (ARGs)
- First-author publications in peer-reviewed journals
- Academic curriculum vitae
- Contact information

## Research Focus

- Environmental antimicrobial resistance (AMR) and antibiotic resistance genes (ARGs) in soil and water systems
- Antibiotic biodegradation and microbial functional ecology
- Nanoplastics and microbe-pollutant interactions
- Biochar-based mitigation strategies
- Stable isotope probing (DNA-SIP), metagenomics, high-throughput qPCR, and epicPCR

## Technical Details

This website is built using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template, a Jekyll-based template for academic websites hosted on GitHub Pages.

### Running Locally

To preview the website locally:

1. Clone the repository
2. Install dependencies: `sudo apt install ruby-dev ruby-bundler nodejs`
3. Run `bundle install` to install ruby dependencies
4. Run `jekyll serve -l -H localhost` to generate and serve the site at `localhost:4000`
